# 🛒 Ecommerce OpenEnv Environment

This project simulates a real-world ecommerce checkout flow for AI agents.

## 🚀 Features
- Add items to cart
- Apply discount coupon
- Checkout and payment flow
- Multi-step task environment

## 🧠 Tasks
- Easy: Complete purchase
- Medium: Apply coupon + purchase
- Hard: Full correct sequence

## ⚙️ API Endpoints

- `/reset` → Reset environment
- `/step` → Take action
- `/state` → Current state
- `/tasks` → Task definitions
- `/grader` → Score evaluation
- `/baseline` → Run baseline agent
